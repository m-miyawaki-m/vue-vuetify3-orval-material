# E2E シナリオ表

## 元デシジョンテーブルとシナリオの対応

| DT ケース | 条件 | 対応シナリオ |
|-----------|------|-------------|
| BQ-1, FP-1 | 条件なし | S1 |
| BQ-2, FP-2 | keyword のみ | S2 |
| BQ-3, FP-3 | category のみ | S3 |
| BQ-4, FP-4 | inStock のみ | S4 |
| BQ-8, FP-8 | 全条件 | S5 |
| FP-9      | 一致なし | S6 |
| FP-10/11  | ページネーション | S7 |
| -         | 詳細→メモ登録フロー | S8 |

---

## シナリオ一覧

### S1: 条件なしで検索 → 全件表示

```
前提: アプリが起動している
操作:
  1. 検索画面 (/search) を開く
  2. キーワード未入力のまま「検索」ボタンを押す
期待結果:
  - /products に遷移する
  - 検索条件チップに「条件なし（全件）」が表示される
  - 商品一覧が表示される（件数 > 0）
```

### S2: キーワードのみで検索 → 一致商品のみ表示

```
前提: アプリが起動している
操作:
  1. 検索画面 (/search) を開く
  2. キーワード入力欄に「緑茶」と入力する
  3. 「検索」ボタンを押す
期待結果:
  - /products?q=緑茶 に遷移する
  - 検索条件チップに「緑茶」が表示される
  - 表示される商品名に「緑茶」が含まれる
```

### S3: カテゴリのみで検索 → カテゴリ一致商品のみ表示

```
前提: アプリが起動している
操作:
  1. 検索画面 (/search) を開く
  2. 「絞り込み」ボタンを押す
  3. ダイアログでカテゴリ「食品」を選択する
  4. ダイアログを閉じる
  5. 「検索」ボタンを押す
期待結果:
  - /products?category=食品 に遷移する
  - 検索条件チップに「食品」が表示される
  - 表示される商品のカテゴリがすべて「食品」
```

### S4: 在庫ありのみで検索 → 在庫あり商品のみ表示

```
前提: アプリが起動している
操作:
  1. 検索画面 (/search) を開く
  2. 「絞り込み」ボタンを押す
  3. 「在庫あり」トグルをONにする
  4. ダイアログを閉じる
  5. 「検索」ボタンを押す
期待結果:
  - /products?inStock=true に遷移する
  - 検索条件チップに「在庫あり」が表示される
  - 表示される商品に「在庫なし」チップがない
```

### S5: 全条件で検索 → 複合絞り込み結果を表示

```
前提: アプリが起動している
操作:
  1. 検索画面 (/search) を開く
  2. キーワード入力欄に「茶」と入力する
  3. 「絞り込み」ボタンを押す
  4. カテゴリ「食品」を選択 + 「在庫あり」トグルをON
  5. ダイアログを閉じる
  6. 「検索」ボタンを押す
期待結果:
  - /products?q=茶&category=食品&inStock=true に遷移する
  - 検索条件チップが3つ表示される（茶 / 食品 / 在庫あり）
  - 表示商品がすべて「食品」かつ「在庫あり」
```

### S6: 一致しないキーワードで検索 → 0件メッセージ表示

```
前提: アプリが起動している
操作:
  1. 検索画面 (/search) を開く
  2. キーワードに「存在しない商品XYZ」と入力する
  3. 「検索」ボタンを押す
期待結果:
  - /products に遷移する
  - 商品カードが表示されない
  - 「見つかりませんでした」等のメッセージが表示される
```

### S7: ページネーション → 2ページ目に移動

```
前提: 商品が6件以上ある（pageSize=5）
操作:
  1. 条件なしで検索して /products を開く
  2. ページネーションの「2」をクリックする
期待結果:
  - 表示商品が切り替わる（1ページ目と異なる商品）
  - ページネーションの「2」がアクティブになる
```

### S8: 詳細 → メモ入力 → 登録 → 一覧でステータス確認

```
前提: 商品一覧が表示されている
操作:
  1. 商品一覧で「未入力」チップがある商品の「詳細を見る」をクリックする
  2. 詳細画面でメモ入力欄に「テストメモ」と入力する
  3. フッターの「登録」ボタンを押す
  4. スナックバーで「登録しました」を確認する
  5. ブラウザバックで一覧に戻る
期待結果:
  - 対象商品のチップが「入力済み」に変わっている
```
