# 画面項目定義書

## 凡例

| 記号 | 意味 |
|------|------|
| ◎ | 必須 |
| ○ | 任意 |
| R | 読み取り専用 |
| - | 非表示 or 該当なし |

---

## 1. 検索ページ (`/search`)

**レイアウト**: MainLayout  
**タイトル**: 商品検索

### 入力項目

| 項目名 | コンポーネント | 必須 | 初期値 | 備考 |
|--------|---------------|------|--------|------|
| キーワード | v-text-field | ○ | '' | 部分一致（name / description） |
| カテゴリ | ProductFilterDialog > v-chip-group | ○ | '' | 食品 / 電子機器 / ファッション / 家具 / スポーツ |
| 在庫ありのみ | ProductFilterDialog > v-switch | ○ | false | true の場合 inStock=true でフィルタ |

### 操作

| ボタン | 処理 |
|--------|------|
| 絞り込み | ProductFilterDialog を開く |
| 検索（フッター） | buildSearchQuery() → `/products?q=...` へ遷移 |
| 読み取り（フッター） | `/scanner` へ遷移 |
| 条件チップ × | 該当フィルタをリセット |

### 表示項目

| 項目名 | 条件 |
|--------|------|
| SearchConditionChips | category または inStockOnly が設定されている場合 |

---

## 2. 商品一覧ページ (`/products`)

**レイアウト**: MainLayout  
**タイトル**: 検索結果

### 表示項目

| 項目名 | コンポーネント | 内容 |
|--------|---------------|------|
| 検索条件チップ | SearchConditionChips | q / category / inStock をチップ表示 |
| 総件数 | v-text | 「XX件」フィルタ後の総件数 |
| オフライン通知 | v-chip | API 失敗時のみ表示（オフラインモード） |
| ローディング | v-progress-linear | API 呼び出し中 |
| 商品カード | ProductCard × N | 1ページ5件 |
| ページネーション | v-pagination | 下部固定 |

### ProductCard 表示項目

| 項目名 | 型 | 備考 |
|--------|-----|------|
| 商品名 | string | |
| カテゴリ | string | |
| 価格 | number | 円表示 |
| 在庫状況 | boolean | 在庫あり / 在庫なし チップ |
| 説明 | string | 省略表示 |

### 操作

| 操作 | 処理 |
|------|------|
| カードクリック | `/detail/:id` へ遷移 |
| ページ変更 | currentPage 更新 → API 再呼び出し |

## 3. 詳細ページ (`/detail/:id`)

**レイアウト**: SubLayout  
**タイトル**: 商品名（取得後に動的設定）

### フッター

| 要素 | 表示条件 | 処理 |
|------|---------|------|
| エラー履歴ボタン（左） | errorHistory.length > 0 | エラー履歴ボトムシートを開く |
| 登録ボタン（中央） | 常時 | 確認ダイアログを開く |

---

### 3-1. 基本情報タブ

#### 商品情報カード

| 項目名 | 型 | 備考 |
|--------|-----|------|
| 商品名 | string | R |
| カテゴリ | string | R |
| 在庫状況チップ | boolean | R / 在庫あり(success) / 在庫なし(error) |
| 説明 | string | R |

#### 入力カード

| 項目名 | コンポーネント | 必須 | 選択肢 | 備考 |
|--------|---------------|------|--------|------|
| ロケーション | SelectPickerField | ◎ | A棚-1 / A棚-2 / B棚-1 / B棚-2 / C棚-1 / C棚-2 | 未選択時バリデーションエラー |
| グループ | SelectPickerField | ◎ | グループA / グループB / グループC | 未選択時バリデーションエラー |
| メモ | BarcodeInputField | ○ | - | バーコードスキャン or テキスト直接入力 |

---

### 3-2. エラー一覧タブ

**バッジ**: 未対応件数を赤バッジで表示

#### エラーカード（1件ごと）

| 項目名 | コンポーネント | 編集可否 | 備考 |
|--------|---------------|---------|------|
| タイトル | テキスト | R | |
| 対応状況チップ | v-chip | R | 未対応(error) / 対応済み(success) |
| 数量 | v-btn(-) + テキスト + v-btn(+) | ○ | 対応済み時は無効化 / 0未満不可 |
| コメント | v-textarea | ○ | 対応済み時は無効化 / 自動拡張 |
| 対応済みにするボタン | v-btn | - | 未対応時のみ表示 |
| 未対応に戻すボタン | v-btn | - | 対応済み時のみ表示 |

---

### 3-3. 確認ダイアログ

| 項目 | 内容 |
|------|------|
| タイトル | 登録確認 |
| メッセージ | 入力内容を登録します。よろしいですか？ |
| ボタン | キャンセル / OK |

---

### 3-4. エラー履歴ボトムシート

| 項目名 | 内容 |
|--------|------|
| タイトル | エラー履歴 |
| リスト | field / message / timestamp の一覧（新着順） |
| 履歴クリアボタン | errorHistory をすべて削除 |
| 上限 | settingsStore.errorHistoryLimit 件（デフォルト100件） |

## 4. スキャナーページ (`/scanner`)

**レイアウト**: フルスクリーン（v-layout 直下）  
**背景**: 黒

### 表示項目

| 項目名 | 条件 | 内容 |
|--------|------|------|
| タイトル | 常時 | scannerStore.title ?? 'バーコード スキャン' |
| フラッシュライトボタン | 常時 | トーチ ON/OFF（非対応デバイスは無効化） |
| カメラ映像 | 常時 | video 要素 |
| スキャン枠 | 常時 | コーナー + スキャンライン |
| カメラエラー | カメラ起動失敗時 | エラーメッセージ |
| DEV モック入力 | `import.meta.env.DEV` のみ | テキスト入力 + 確定ボタン（オレンジ枠） |
| スキャン履歴リスト | continuous モードのみ | スキャン済みバーコード一覧 |
| 完了ボタン | continuous モードのみ | 「完了（N件）」 |

### 操作

| 操作 | モード | 処理 |
|------|--------|------|
| バーコード検出 | single | complete([result]) → back() |
| バーコード検出 | continuous | results に追加 |
| 完了ボタン | continuous | complete([...results]) → back() |
| × ボタン（履歴） | continuous | results から削除 |
| クリアボタン | continuous | results を空に |
| ← 戻るボタン | 両方 | cancel() → back() |

## 5. メインメニューページ (`/menu`)

**レイアウト**: MainLayout  
**タイトル**: メインメニュー

### 表示項目

| 項目名 | コンポーネント | 内容 |

| ローディング | v-progress-linear | fetchMenu() 実行中 |

### メニュー項目（main-menu.json）

| ID | ラベル | サブメニュー数 |
|----|--------|--------------|
| order | 受注管理 | 3 |
| inventory | 在庫管理 | 3 |
| customer | 顧客管理 | 2 |
| product | 商品管理 | 2 |
| shipping | 配送管理 | 2 |
| report | 帳票・集計 | 2 |
| settings | 設定 | 1 |

**レイアウト**: MainLayout  
**タイトル**: 設定

### 入力項目

| セクション | 項目名 | コンポーネント | 備考 |
